1. Blog personnel
Contexte
Créer un blog personnel où les utilisateurs peuvent publier des articles après s’être inscrits et connectés.
Objectifs fonctionnels
•
Formulaire d'inscription et de connexion sécurisés
•
Zone de gestion des articles pour les utilisateurs connectés :
o
Publier un nouvel article
o
Modifier ou supprimer un article existant
•
Affichage des articles sur une page principale avec pagination (par exemple, 5 articles par page)
Contraintes techniques
•
Langages : PHP (sans POO), HTML, CSS, SQL
•
Base de données : Une table users (pour gérer les utilisateurs) et une table articles (pour gérer les articles)
•
Bootstrap : Utilisation pour la structure et le style des pages
•
MCD et MLD : Les stagiaires doivent concevoir un MCD et un MLD avant de créer la base de données
Design et ergonomie
•
Figma : Création d’une maquette avant le développement
•
Responsive Web Design : Le site doit être lisible et navigable sur ordinateur, tablette et smartphone
•
Bootstrap : Utilisation pour la grille responsive et les composants (navbar, formulaire, etc.)
